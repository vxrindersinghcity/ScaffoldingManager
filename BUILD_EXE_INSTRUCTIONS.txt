  HOW TO CREATE WINDOWS .EXE FILE (For Windows Users)
============================================================

If you want a single .exe file that doesn't require Python
installation, follow these steps ON A WINDOWS COMPUTER:


üìã PREREQUISITES
----------------
1. Windows 7 or higher
2. Python 3.8+ installed
   Download: https://www.python.org/downloads/

üîß STEP-BY-STEP INSTRUCTIONS
-----------------------------

STEP 1: Extract Files
  ‚Ä¢ Extract the ScaffoldingManager_Complete.zip
  ‚Ä¢ You should have these files:
    - scaffolding_manager.py
    - complete_scaffolding_manager.html
    - START_MANAGER.bat
    - README.txt
    - requirements.txt

STEP 2: Open Command Prompt
  ‚Ä¢ Press Windows Key + R
  ‚Ä¢ Type: cmd
  ‚Ä¢ Press Enter

STEP 3: Navigate to Folder
  ‚Ä¢ In Command Prompt, type:
    cd C:\path\to\extracted\folder
  ‚Ä¢ Replace with your actual folder path
  ‚Ä¢ Press Enter

STEP 4: Install PyInstaller
  ‚Ä¢ Type: pip install pyinstaller
  ‚Ä¢ Press Enter
  ‚Ä¢ Wait for installation to complete

STEP 5: Build the EXE
  ‚Ä¢ Type this command (all on one line):
    pyinstaller --onefile --windowed --add-data "complete_scaffolding_manager.html;." --name "ScaffoldingManager" scaffolding_manager.py
  
  ‚Ä¢ Press Enter
  ‚Ä¢ Wait 2-5 minutes for build to complete

STEP 6: Find Your EXE
  ‚Ä¢ The .exe file will be in:
    dist\ScaffoldingManager.exe
  ‚Ä¢ This is your standalone executable!


‚úÖ USING THE EXE FILE
---------------------
1. Copy ScaffoldingManager.exe to any folder
2. Double-click to run
3. Your browser will open automatically
4. Keep the console window open
5. Press Ctrl+C to stop

‚öôÔ∏è ADVANCED: Custom Icon

To add a custom icon to your .exe:

üîç TROUBLESHOOTING
------------------

‚ùå "PyInstaller is not recognized"
   ‚Üí Restart Command Prompt after installing
   ‚Üí Or use: python -m PyInstaller instead

‚ùå Build fails with errors
   ‚Üí Make sure all files are in the same folder
   ‚Üí Check Python version (needs 3.8+)
   ‚Üí Try: pip install --upgrade pyinstaller

‚ùå EXE won't run
   ‚Üí Check Windows Defender/antivirus
   ‚Üí Add exception for the .exe file
   ‚Üí Some antivirus programs flag PyInstaller exes

‚ùå "Failed to execute script"
   ‚Üí Ensure complete_scaffolding_manager.html is bundled
   ‚Üí Try rebuilding with the exact command above
